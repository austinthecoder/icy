# Icy

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'icy'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install icy

## Usage

```
Icy.require_tree 'my_dir' # require every ruby file in this directory
```
